# Changelog

All notable changes to "React Native SVG Preview" will be documented in this file.

## [0.1.0] - 2024-12-02

### Added
- Initial release
- Real-time preview of React Native SVG components
- Support for all major SVG elements (Circle, Rect, Path, G, etc.)
- Gradient support (LinearGradient, RadialGradient)
- Text element support
- Zoom control (25% - 400%)
- Background color options (white, black, gray, checkerboard, custom)
- Live update on code changes
